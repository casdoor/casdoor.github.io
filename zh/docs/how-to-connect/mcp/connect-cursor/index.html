е°Ҷе…үж ҮиҝһжҺҘеҲ°MCP

е°ҶCursor IDEиҝһжҺҘеҲ°Casdoorзҡ„MCPжңҚеҠЎеҷЁпјҢд»ҘдҫҝжӮЁиғҪд»Һзј–иҫ‘еҷЁдёӯз®ЎзҗҶеә”з”ЁгҖҒз”ЁжҲ·е’Ңиө„жәҗгҖӮ

е…ҲеҶіжқЎд»¶вҖӢ

  • дёҖдёӘжӯЈеңЁиҝҗиЎҢзҡ„Casdoorе®һдҫӢпјҲз”ҹдә§зҺҜеўғжҺЁиҚҗдҪҝз”ЁHTTPSпјү
  • е·ІеңЁжӮЁзҡ„и®Ўз®—жңәдёҠе®үиЈ…зҡ„bCursor IDE
  • еҜ№жӮЁзҡ„Casdoorе®һдҫӢе…·жңүз®ЎзҗҶе‘ҳи®ҝй—®жқғйҷҗпјҢд»ҘдҫҝеҲӣе»әеә”з”Ё

жӯҘйӘӨ1пјҡеңЁCasdoorдёӯеҲӣе»әдёҖдёӘеә”з”ЁвҖӢ

дёәCursorзҡ„OAuthеҲӣе»әдёҖдёӘCasdoorеә”з”Ёпјҡ

  1. зҷ»еҪ•жӮЁзҡ„Casdoorз®ЎзҗҶйқўжқҝ

  2. еҜјиҲӘиҮіеә”з”Ёе№¶зӮ№еҮ»ж·»еҠ 

  3. дҪҝз”Ёд»ҘдёӢи®ҫзҪ®й…ҚзҪ®еә”з”ЁзЁӢеәҸпјҡ

    • еҗҚз§°пјҡcursor-mcpпјҲжҲ–жӮЁеҒҸеҘҪзҡ„еҗҚз§°пјү

    • жҳҫзӨәеҗҚз§°пјҡе…үж ҮIDE MCPе®ўжҲ·з«Ҝ

    • з»„з»ҮпјҡйҖүжӢ©жӮЁзҡ„з»„з»Ү

    • йҮҚе®ҡеҗ‘ URIпјҡж·»еҠ д»ҘдёӢ OAuth еӣһи°ғ URLпјҡ

      http://127.0.0.1:*/callback
      http://localhost:*/callback
жҸҗзӨә

йҖҡй…Қз¬Ұ * е…Ғи®ё Cursor дҪҝз”Ёд»»дҪ•еҸҜз”Ёз«ҜеҸЈиҝӣиЎҢ OAuth еӣһи°ғгҖӮ

  1. жҺҲжқғзұ»еһӢпјҡеҗҜз”Ё жҺҲжқғз ҒпјҢе№¶еҸҜйҖүеҗҜз”Ё еҲ·ж–°д»ӨзүҢ

  2. еҗҜз”ЁPKCEпјҡеӢҫйҖүжӯӨйҖүйЎ№д»Ҙеўһејәе®үе…ЁжҖ§

  3. Token Format: JWT (recommended)

  4. пјҲеҸҜйҖүпјүеә”з”Ёзұ»еһӢпјҡи®ҫзҪ®дёә Agent

  5. пјҲеҸҜйҖүпјүзұ»еҲ«пјҡдёәдҫҝдәҺз»„з»ҮпјҢи®ҫзҪ®дёә MCP

    ::info иҜ·еҸӮйҳ…еә”з”Ёзұ»еҲ«дәҶи§Јзұ»еҲ«е’Ңзұ»еһӢйҖүйЎ№гҖӮ

:::

  1. еҚ•еҮ»дҝқеӯҳпјҢе№¶и®°дёӢдёӢдёҖжӯҘжүҖйңҖзҡ„е®ўжҲ·з«Ҝ IDгҖӮ

жӯҘйӘӨ2пјҡй…ҚзҪ®е…үж ҮMCPи®ҫзҪ®вҖӢ

е…үж ҮеҸҜйҖҡиҝҮе…¶и®ҫзҪ®ж”ҜжҢҒMCPжңҚеҠЎеҷЁй…ҚзҪ®гҖӮ жӮЁеҸҜд»ҘйҖҡиҝҮз”ЁжҲ·з•ҢйқўжҲ–зӣҙжҺҘзј–иҫ‘й…ҚзҪ®ж–Үд»¶жқҘеҜ№е…¶иҝӣиЎҢй…ҚзҪ®гҖӮ

йҖүйЎ№ AпјҡдҪҝз”Ёе…үж Үи®ҫзҪ® UIвҖӢ

  1. жү“ејҖ Cursor IDE
  2. еүҚеҫҖи®ҫзҪ®пјҲmacOS дёҠдёә Cmd+,пјҢWindows/Linux дёҠдёә Ctrl+,пјү
  3. еңЁи®ҫзҪ®жҗңзҙўж ҸдёӯжҗңзҙўвҖңMCPвҖқ
  4. зӮ№еҮ»еңЁsettings.jsonдёӯзј–иҫ‘д»Ҙжү“ејҖMCPй…ҚзҪ®ж–Үд»¶

йҖүйЎ№BпјҡзӣҙжҺҘзј–иҫ‘ж–Үд»¶вҖӢ

MCPй…ҚзҪ®ж–Үд»¶зҡ„дҪҚзҪ®еҸ–еҶідәҺжӮЁзҡ„ж“ҚдҪңзі»з»ҹпјҡ

  • macOS: ~/Library/Application Support/Cursor/User/mcp.json
  • Windows: %APPDATA%\Cursor\User\mcp.json
  • Linux: ~/.config/Cursor/User/mcp.json

ж·»еҠ жӮЁзҡ„Casdoor MCPжңҚеҠЎеҷЁй…ҚзҪ®пјҡ

{
  "mcpServers": {
    "casdoor": {
      "url": "https://your-casdoor.com/api/mcp"
    }
  }
}

жӣҝжҚўд»ҘдёӢеҚ дҪҚз¬Ұпјҡ

  • your-casdoor.com вҶ’ жӮЁзҡ„Casdoorе®һдҫӢеҹҹеҗҚ
еӨҮжіЁ

е…үж ҮеҺҹз”ҹж”ҜжҢҒйҖҡиҝҮurlеӯ—ж®өиҝһжҺҘеҹәдәҺHTTPзҡ„иҝңзЁӢMCPжңҚеҠЎеҷЁгҖӮ OAuth жөҒзЁӢе°ҶиҮӘеҠЁеӨ„зҗҶвҖ”вҖ”Cursor дјҡжү“ејҖжӮЁзҡ„жөҸи§ҲеҷЁпјҢдҪҝз”Ёеӣһи°ғ URL cursor://anysphere.cursor-mcp/oauth/callback е®ҢжҲҗиә«д»ҪйӘҢиҜҒгҖӮ

й…ҚзҪ®дҪңз”ЁеҹҹвҖӢ

й…ҚзҪ®дҪңз”Ёеҹҹ еёёз”ЁиҢғеӣҙеҢ…жӢ¬пјҡ

  • read:application - жҹҘзңӢеә”з”Ё
  • write:application - еҲӣе»әгҖҒжӣҙж–°гҖҒеҲ йҷӨеә”з”ЁзЁӢеәҸ
  • read:user - жҹҘзңӢз”ЁжҲ·
  • write:user - еҲӣе»әгҖҒжӣҙж–°гҖҒеҲ йҷӨз”ЁжҲ·
  • openid profile email - еҹәжң¬з”ЁжҲ·дҝЎжҒҜпјҲOAuthжүҖйңҖпјү

жңүе…іеҸҜз”ЁдҪңз”Ёеҹҹзҡ„е®Ңж•ҙеҲ—иЎЁпјҢиҜ·еҸӮйҳ…жҺҲжқғдёҺдҪңз”ЁеҹҹгҖӮ

жӯҘйӘӨ3пјҡйҮҚж–°еҠ иҪҪCursorвҖӢ

дҝқеӯҳй…ҚзҪ®еҗҺпјҡ

  1. йҮҚж–°еҠ иҪҪ Cursor IDEпјҲCmd+Shift+P жҲ– Ctrl+Shift+P вҶ’ вҖңејҖеҸ‘иҖ…пјҡйҮҚж–°еҠ иҪҪзӘ—еҸЈвҖқпјү
  2. жҲ–иҖ…пјҢе®Ңе…ЁйҮҚеҗҜCursor

Cursor е°ҶиҮӘеҠЁжЈҖжөӢе№¶еҠ иҪҪж–°зҡ„ MCP жңҚеҠЎеҷЁй…ҚзҪ®гҖӮ

жӯҘйӘӨ 4пјҡе®ҢжҲҗ OAuth жөҒзЁӢвҖӢ

еҪ“ Cursor йҰ–ж¬ЎиҝһжҺҘеҲ°жӮЁзҡ„ Casdoor MCP жңҚеҠЎеҷЁж—¶пјҡ

  1. е…үж Үе°ҶиҮӘеҠЁжү“ејҖжӮЁзҡ„й»ҳи®ӨзҪ‘з»ңжөҸи§ҲеҷЁ
  2. жӮЁе°ҶзңӢеҲ°Casdoorзҷ»еҪ•йЎөйқўпјҲеҰӮжһңе°ҡжңӘзҷ»еҪ•пјү
  3. зҷ»еҪ•еҗҺпјҢжӮЁе°ҶзңӢеҲ°дёҖдёӘеҗҢж„ҸеұҸе№•пјҢиҰҒжұӮжӮЁжҺҲжқғCursor
  4. еҗҢж„ҸеұҸе№•дјҡжҳҫзӨәжүҖиҜ·жұӮзҡ„дҪңз”ЁеҹҹпјҲжқғйҷҗпјү
  5. зӮ№еҮ»жҺҲжқғд»ҘжҺҲдәҲи®ҝй—®жқғйҷҗ
  6. жӮЁзҡ„жөҸи§ҲеҷЁе°ҶйҮҚе®ҡеҗ‘еҲ° http://127.0.0.1:<0>/callback е№¶жҳҫзӨәжҲҗеҠҹж¶ҲжҒҜ
  7. иҝ”еӣһе…үж Ү - иҝһжҺҘзҺ°е·Іе»әз«Ӣ
жҸҗзӨә

OAuth д»ӨзүҢз”ұ MCP OAuth иҫ…еҠ©зЁӢеәҸе®үе…ЁеӯҳеӮЁгҖӮ йҷӨйқһжӮЁж’Өй”Җд»ӨзүҢжҲ–жӣҙж”№дҪңз”ЁеҹҹпјҢеҗҰеҲҷж— йңҖйҮҚж–°жҺҲжқғгҖӮ

жӯҘйӘӨ5пјҡйӘҢиҜҒиҝһжҺҘвҖӢ

дҪҝз”ЁCursorзҡ„AIеҠҹиғҪдёҺCasdoorдәӨдә’пјҢжөӢиҜ•иҝһжҺҘпјҡ

**еңЁCursorиҒҠеӨ©дёӯе°қиҜ•зҡ„зӨәдҫӢжҸҗзӨәпјҡ

  • вҖңдҪҝз”ЁCasdoor MCPжңҚеҠЎеҷЁпјҢеҲ—еҮәжүҖжңүеә”з”ЁвҖқ
  • вҖңеҗ‘жҲ‘еұ•зӨәе…ідәҺCasdoorдёӯеҗҚдёәвҖҳmy-appвҖҷзҡ„еә”з”ЁзЁӢеәҸзҡ„иҜҰз»ҶдҝЎжҒҜвҖқ
  • еңЁз»„з»ҮвҖҳmy-orgвҖҷдёӯеҲӣе»әеҗҚдёәвҖҳtest-appвҖҷзҡ„ж–°Casdoorеә”з”Ё

еҺҹеӣ пјҡMCP жңҚеҠЎеҷЁ URL еҸҜиғҪдёҚжӯЈзЎ®жҲ–ж— жі•и®ҝй—®гҖӮ

и§ЈеҶіж–№жЎҲпјҡ

  • йӘҢиҜҒжӮЁзҡ„ mcp.json дёӯзҡ„ URL жҳҜеҗҰжӯЈзЎ®
  • зЎ®дҝқжӮЁзҡ„Casdoorе®һдҫӢжӯЈеңЁиҝҗиЎҢдё”еҸҜи®ҝй—®
  • жЈҖжҹҘHTTPS/HTTPдёҚеҢ№й…ҚпјҲз”ҹдә§зҺҜеўғдҪҝз”ЁHTTPSпјү

й—®йўҳпјҡOAuthжңҹй—ҙеҮәзҺ°вҖңйҮҚе®ҡеҗ‘URIдёҚеҢ№й…ҚвҖқй”ҷиҜҜвҖӢ

еҺҹеӣ пјҡеӣһи°ғURLдёҺCasdoorдёӯй…ҚзҪ®зҡ„йҮҚе®ҡеҗ‘URIдёҚеҢ№й…ҚгҖӮ

й—®йўҳпјҡжөҸи§ҲеҷЁжҺ§еҲ¶еҸ°дёӯзҡ„вҖңCORS й”ҷиҜҜвҖқвҖӢ

еҺҹеӣ пјҡи·Ёжәҗиө„жәҗе…ұдә«пјҲCORSпјүйҷҗеҲ¶гҖӮ

и§ЈеҶіж–№жЎҲпјҡ

  • Casdoorзҡ„MCPз«ҜзӮ№еә”иҮӘеҠЁеӨ„зҗҶжң¬ең°дё»жңәжқҘжәҗзҡ„CORS
  • еҰӮжһңжӮЁдҪҝз”Ёзҡ„жҳҜиҮӘе®ҡд№үеҹҹеҗҚпјҢиҜ·зЎ®дҝқеңЁCasdoorдёӯжӯЈзЎ®й…ҚзҪ®дәҶCORS

й—®йўҳпјҡдҪҝз”ЁAIеҠҹиғҪж—¶еҮәзҺ°вҖңinsufficient_scopeвҖқй”ҷиҜҜвҖӢ

еҺҹеӣ пјҡжүҖиҜ·жұӮзҡ„ж“ҚдҪңйңҖиҰҒдёҖдёӘжңӘиў«жҺҲдәҲзҡ„дҪңз”ЁеҹҹгҖӮ

и§ЈеҶіж–№жЎҲпјҡ

  • еңЁжӮЁзҡ„mcp.jsonдёӯжӣҙж–°OAUTH_SCOPESд»ҘеҢ…еҗ«жүҖйңҖзҡ„дҪңз”Ёеҹҹ
  • зӨәдҫӢпјҡеҰӮжһңжӮЁжғіеҲӣе»әжҲ–дҝ®ж”№еә”з”ЁзЁӢеәҸпјҢиҜ·ж·»еҠ write:application
  • йҮҚж–°еҠ иҪҪе…үж Үе№¶йҮҚж–°жҺҲжқғпјҢд»ҘиҺ·еҸ–еёҰжңүжӣҙж–°дҪңз”Ёеҹҹзҡ„ж–°д»ӨзүҢ

и§ЈеҶіж–№жЎҲпјҡ

  • еҰӮжһңжӮЁеңЁжӯҘйӘӨ1дёӯеҗҜз”ЁдәҶеҲ·ж–°д»ӨзүҢжҺҲжқғзұ»еһӢпјҢMCP OAuthеҠ©жүӢе°ҶиҮӘеҠЁеҲ·ж–°иҝҮжңҹзҡ„д»ӨзүҢ
  • еҗҰеҲҷпјҢжӮЁйңҖиҰҒйҖҡиҝҮйҮҚж–°еҠ иҪҪ Cursor жқҘйҮҚж–°жҺҲжқғ

еҺҹеӣ пјҡCursorеҸҜиғҪжңӘеҠ иҪҪMCPжңҚеҠЎеҷЁпјҢжҲ–OAuthе°ҡжңӘе®ҢжҲҗгҖӮ

и§ЈеҶіж–№жЎҲпјҡ

  • еңЁжӮЁзҡ„жҸҗзӨәдёӯжҳҺзЎ®жҸҗеҸҠвҖңдҪҝз”ЁCasdoor MCPжңҚеҠЎеҷЁвҖқ
  • зЎ®дҝқжӮЁе·Іе®ҢжҲҗз¬¬4жӯҘдёӯзҡ„OAuthжөҒзЁӢ

е®үе…ЁжіЁж„ҸдәӢйЎ№вҖӢ

  • PKCEпјҲд»Јз ҒдәӨжҚўиҜҒжҳҺеҜҶй’ҘпјүпјҡеңЁжӮЁзҡ„Casdoorеә”з”Ёдёӯе§Ӣз»ҲеҗҜз”ЁPKCEпјҢд»Ҙеўһејәе®үе…ЁжҖ§
  • дҪңз”ЁеҹҹпјҡйҒөеҫӘжңҖе°ҸжқғйҷҗеҺҹеҲҷвҖ”вҖ”д»…жҺҲдәҲ Cursor е®һйҷ…жүҖйңҖзҡ„иҢғеӣҙ
  • д»ӨзүҢеӯҳеӮЁпјҡMCP OAuthеҠ©жүӢдјҡе°Ҷд»ӨзүҢе®үе…Ёең°еӯҳеӮЁеңЁжӮЁзі»з»ҹзҡ„й’ҘеҢҷдёІдёӯ
  • HTTPSпјҡеҜ№дәҺз”ҹдә§зҺҜеўғзҡ„Casdoorе®һдҫӢпјҢе§Ӣз»ҲдҪҝз”ЁHTTPSд»ҘдҝқжҠӨOAuthжөҒзЁӢ
  • д»ӨзүҢж’Өй”ҖпјҡжӮЁеҸҜд»ҘеңЁCasdoorзҡ„з®ЎзҗҶйқўжқҝдёӯзҡ„вҖҳд»ӨзүҢвҖҷдёӢж’Өй”Җи®ҝй—®д»ӨзүҢ
  • д»Јз Ғе®ЎжҹҘпјҡеңЁжү§иЎҢдёҺCasdoorдәӨдә’зҡ„AIз”ҹжҲҗд»Јз Ғд№ӢеүҚпјҢеҠЎеҝ…иҝӣиЎҢе®ЎжҹҘ
