Azure Communication Services email

Use Azure Communication Services for sending email. Complete:

  • メール通信サービスの作成と管理
  • 無料のAzure管理ドメインを取得する または カスタムドメインを追加する
  • ドメインを接続する

Copy your Endpoint and Private Key from the resource.

Configure the provider in Casdoor​

Providers → Add. Set Category to Email, Type to the option for Azure ACS. Map:

CasdoorAzure ACS
差出人アドレスVerified sender (must use a verified domain)
シークレットキープライベートキー
ホストエンドポイント
注

From Address must use a verified email domain in ACS.
