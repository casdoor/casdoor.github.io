Face ID

Visão Geral​

Incorporamos agora o login com Face ID no Casdoor utilizando o face-api.js.

Método de ativação​

Adicione a opção Face ID nos itens de Conta da organização​

Gerenciamento de Usuários -> Organizações -> Escolha uma organização -> Localize a seção Itens de Conta e incorpore o Face ID

Depois, você encontrará a opção Face ID em Usuário, onde os usuários podem fazer upload de seus dados faciais para serem usados no login​

Gerenciamento de Usuários -> Usuários -> Escolha um usuário -> Encontre Face Ids, e adicione dados faciais. Você pode adicionar até 5 entradas de dados faciais, e pode dar a cada dado facial um nome personalizado.

Terceiro passo: Incorpore o Face ID como uma opção de login na seção Métodos de login do aplicativo​

Identidade -> Aplicações -> Escolha uma aplicação -> Vá para a seção Métodos de login e incorpore o Face ID como uma opção de login.

Finalmente, você pode fazer login usando o método Face ID na página de login​

  1. Na página de login, selecione o método de login Face ID.
  2. Digite o nome de usuário, clique em Entrar com Face ID.
  3. Uma vez que você conceda permissão para acessar sua câmera, você poderá fazer login usando o Face ID.
