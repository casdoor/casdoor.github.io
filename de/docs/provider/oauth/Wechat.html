Visit WeChat developer platform, and register as a developer. After your web application or your mobile application is approved, then you get your App Id and App Secret.

The WeChat provider offers two different sets of keypairs:

  • The second keypair (Client ID 2, Client Secret 2) is for WeChat Media Platform (ÂæÆ‰ø°ÂÖ¨‰ºóÂπ≥Âè∞), it's only for the inside-WeChat-app login scenario. It allows the user to log in with WeChat built-in browser inside WeChat mobile APP, it will jump to your WeChat Official Account (ÂæÆ‰ø°ÂÖ¨‰ºóÂè∑) to log in. It's notable that in the mobile scenario, WeChat itself doesn't support logging in outside of WeChat APP, like in other mobile browsers (H5) or APPs. This is a limitation of WeChat instead of Casdoor.

If you fill in the second keypair (Client ID 2, Client Secret 2) and enable the Enable QR code switch, when the user clicks on the WeChat button to log in, Casdoor will first ask the user to follow the WeChat official account (ÂæÆ‰ø°ÂÖ¨‰ºóÂè∑), then continue the login process. It's notable that this is only available in PC login scenario because a mobile phone cannot scan the QR code by itself. Casdoor will automatically skip this step when used in mobile scenario (aka the WeChat built-in browser inside WeChat mobile APP).

tip

We recommend setting the two key sets at the same time, and linking your WeChat Open Platform (ÂæÆ‰ø°ÂºÄÊîæÂπ≥Âè∞) account and WeChat Media Platform (ÂæÆ‰ø°ÂÖ¨‰ºóÂπ≥Âè∞) account together inside WeChat Open Platform (ÂæÆ‰ø°ÂºÄÊîæÂπ≥Âè∞). So a WeChat user logged-in through PC and mobile can be recognized as the same user in Casdoor.

note

Due to the limitations of WeChat OAuth, there is currently no way to log in via WeChat in a 3rd-party mobile APP or in a mobile browser other than WeChat APP. The mobile login must happen inside WeChat APP for now.

For more detailed information, please visit WeChat Open Platform.
