什么是Casdoor？​

Casdoor是基于 OAuth 2.0 / OIDC 的 UI 集中身份验证 / 单点登录(SSO) 平台。

Casdoor由 Casbin 使用 Golang 和 JavaScript 开发。

Casdoor旨在提供一个强大的身份认证平台来处理用户管理问题。

为什么要使用Casdoor？​

  • ⚡️ 强大的功能和极简的配置。 支持 登录和传统的用户名/密码登录，Casdoor帮你只需几个步骤即可实施您的用户管理系统。

  • 💥 前后端分离架构。 Casdoor提供可随心所欲的网页界面并支持高并发。

  • 🥂 活跃的开源社区。 Casdoor是开源的，并有专业团队积极维护中。

  • 🌐 多语言支持。 Casdoor支持多种语言，包括中文、英文、法文、以及更多语言。

如何使用Casdoor？​

您可以通过阅读详细的文档Casdoor API and ，设置您自己的casdoor。 或者使用 Casdoor SDK 来实现您自己的应用程序。
