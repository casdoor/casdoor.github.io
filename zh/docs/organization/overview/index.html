概览

本组织是Cassdoor的基本单位，它管理用户和应用程序。 如果用户登录到一个组织，他可以访问所有属于该组织的应用程序，而不必再次登录。

在 应用程序 和 提供商中， 选择一个组织很重要，它决定一个用户是否能够使用特定的提供商访问该应用程序。

我们也可以在 Cassdoor 中设置LDAP 以了解更多详情，请查看 LDAP
