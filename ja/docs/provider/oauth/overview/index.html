概要

Casdoorは、他のOAuthアプリケーションをサインインメソッドとして使用できます。

現在、Casdoorは多くのOAuthアプリケーションプロバイダをサポートしています。 プロバイダのアイコンは、Casdoorに追加するとログインページとサインアップページに表示されます。 Casdoorがサポートしているプロバイダーは次のとおりです。

GoogleGitHubFacebookTwitterLinkedInWeiboWeChatTencent QQDingtalkGiteeEメールアドレスSMS
✅✅✅🚧✅✅✅✅✅✅✅✅

私たちは、サードパーティのサービスを申請し、それをキャスドアに追加する方法を紹介します。

開発者になるために申請する​

この前に、あなたが理解する必要があるいくつかの一般的な概念があります。

  • Redirect Url, Redirect address after authentication, fill in your application address, such as https://forum.casbin.com/
  • スコープ、基本プロファイル、メールアドレス、投稿等のユーザーに付与される許可。
  • ClientId/AppId, ClientKey/AppSecret, This is the most important information, and it is what you need to get after you apply for a developer account, can not share the key/secret with anyone.

Twitter(still working🚧)​

Twitterのアプリケーションステップはやや面倒で、公式の制限は少し厳しくなっています。 他のサードパーティのプラットフォームよりも開発者アカウントへの申請が難しいかもしれません。

Developer Portalにアクセスし、アカウントをお持ちでない場合は登録します。 Twitterは開発者アカウントに何を申請しているかを知る必要があります。 慎重にそれを記入しなければなりません、そうでなければそれは通過しません。

アプリケーションが承認された後、アプリケーションを作成し、コールバックアドレスとその他の情報を入力します。 設定される2つのことを行う必要があります。これは 認証設定 セクションです。

  • 手動で 3本足のOAuthをオンにすると、Twitterでサインインする、他のアカウントに代わってツイートを投稿することができます。
  • ユーザーのメールアドレスを取得するために、 ****からメールアドレスを要求することを有効にします。

Weibo✔️​

Weiboの開発者アカウントのアプリケーションは難しいことではありませんが、速度は比較的遅く、それは約2-3日かかります。

開発者ウェブサイトにアクセスし、基本的な情報を入力し、長いレビューを待っています...

レビューが承認されると、クライアント ID とクライアント シークレットを取得できます。

WeChat✔​

WeChat 開発者プラットフォームにアクセスし、開発者として登録します。 Webアプリケーションまたはモバイルアプリケーションが承認されると、App IDとApp Secretが取得されます。

詳細については、 WeChat Open Platform をご覧ください。

Tencent QQ✔​

QQ の認証プラットフォームを訪問 - QQ を接続します。

レビューが承認された後、まず 開発者になるに申請する必要があります。 プラットフォームの指示に従って、クライアントIDとクライアントシークレットを取得します。

DingTalk✔​

DingTalk 開発者向けプラットフォーム にアクセスし、DingTalk アカウントを使用してログインします。 プラットフォームに入った後、プラットフォームの指示に従ってクライアントIDとクライアントシークレットを取得します。

詳細については、 DingTalk 開発者ドキュメント をご覧ください。

OAuth プロバイダーを追加​

  1. Casdoorのインデックスページに移動します
  2. トップバーの プロバイダー をクリックしてください。
  3. 追加をクリックすると、リストの上部に新しいプロバイダが表示されます。
  4. 変更するには、新しいプロバイダをクリックしてください
  5. カテゴリ で OAuth を選択します
  6. タイプで必要なOAuthプロバイダを選択してください
  7. 最もインポートされた情報を入力します。 クライアント ID と クライアント シークレット

アプリケーションに適用​

  1. トップバーの アプリケーション をクリックし、1 つのアプリケーションを選択し、編集
  2. プロバイダ追加ボタンをクリックし、追加したプロバイダを選択します
  3. 登録、ログイン、解除など、プロバイダーの権限を変更する
  4. 完了！
